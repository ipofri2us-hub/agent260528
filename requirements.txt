# ============================================================
# 삼성디스플레이 재직자 대상 AI Agent Architecture 1~5일차 실습
# requirements.txt
# ============================================================
#
# 이 파일은 1일차 Prompt / Chain / mini Graph / Day1 Agent v0,
# 2일차 Markdown RAG / Chroma Vector DB / LangGraph RAG Agent,
# 3일차 PostgreSQL 제조 DB/Log Tool / RAG Tool wrapper / MCP 형식 Tool 호출,
# 4일차 Trace 분석 / Contract 검증 / 오류 재현 / Guardrail,
# 5일차 Final MCP Multi-Agent / Trace / 최종 리포트 실습에 사용합니다.
#
# LLM 사용 구조:
# - mock  : API Key 없이 사용하는 기본 교육용 Mock LLM입니다.
# - local : Ollama 로컬 서버를 HTTP API로 호출합니다.
# - cloud : Gemini, OpenAI, Anthropic, NVIDIA 중 하나를 .env에서 선택해 사용합니다.
#
# NVIDIA mistral-nemotron 사용 방식:
# - 프로젝트 기본 구현은 OpenAI-compatible SDK 방식으로 NVIDIA endpoint를 호출합니다.
# - .env 예시:
#     LLM_PROVIDER=cloud
#     CLOUD_LLM_PROVIDER=nvidia
#     NVIDIA_BASE_URL=https://integrate.api.nvidia.com/v1
#     NVIDIA_MODEL_NAME=mistralai/mistral-nemotron
#
# 2일차 RAG 실습에서 Chroma Vector DB를 사용하므로 chromadb를 포함합니다.
# 3일차 PostgreSQL 실습에서는 Docker로 실행한 교육용 PostgreSQL에
# Python으로 접속하기 위해 psycopg2-binary를 사용합니다.
#
# 제외 항목:
# - faiss: Chroma를 사용하므로 FAISS는 사용하지 않습니다.
# - mcp 공식 패키지: 교육용 Minimal MCP 구조 또는 fallback Tool Registry를 사용합니다.
#
# 선택 사항:
# - LangChain ChatNVIDIA 예제를 직접 실행할 때만 별도로 설치합니다.
#   python -m pip install langchain-nvidia-ai-endpoints
#   현재 기본 프로젝트 코드는 OpenAI-compatible SDK로 NVIDIA를 호출하므로 필수는 아닙니다.
# ============================================================
